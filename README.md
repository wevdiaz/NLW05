 ### :hammer_and_wrench: Tecnologias
 
 * Typescript
 * Express
 * Banco de Dados - SQLite3
 * Websocket
 * typeorm
 
 
 ### :gear: Requisitos
 
 Instalar o nodejs 
 
* npm install -> para instalar as dependências necessárias do projeto.
* npm run dev -> para rodar aplicação
